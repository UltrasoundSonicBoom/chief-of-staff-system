# Team Roster

## Chief of Staff Agent
- Mission: overall prioritization, orchestration, daily/weekly briefings, escalation management
- Inputs: Calendar, Things, Notion HQ, signals
- Outputs: top priorities, escalations, operating reviews

## Finance Agent
- Mission: cashflow, debt, billing, renewals, purchase tracking
- Inputs: email, SMS, orders, finance DBs
- Outputs: alerts, reviews, finance summaries

## Family Operations Agent
- Mission: school, academy, homework, prep items, family logistics
- Inputs: school notices, family calendar, reminders
- Outputs: family brief, reminders, prep lists

## Home & Purchasing Agent
- Mission: cleaning, inventory, repeat-buy cycle analysis
- Inputs: order history, reminders, home DBs
- Outputs: maintenance plan, reorder suggestions

## Professional Strategy Agent
- Mission: hospital operations, SNUHmate, automation roadmap
- Inputs: projects, notes, meetings
- Outputs: milestones, blockers, execution proposals

## KRTA Strategy Agent
- Mission: KRTA homepage, committee operations, AI agenda
- Inputs: KRTA HQ, email threads, meetings, docs
- Outputs: initiative updates, committee tracking, agenda drafts

## Venture Studio Agent
- Mission: promote ideas into experiments, PRDs, or projects
- Inputs: Obsidian ideas, project pipeline
- Outputs: opportunity briefs, PRD candidates

## Device / Security / Infra Agent
- Mission: MacBook/Mac mini/Windows/backup/security/storage governance
- Inputs: system checks, asset inventory, storage alerts
- Outputs: infra alerts, health checks, mitigation tasks
