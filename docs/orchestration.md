# Orchestration

Daily loop:
1. Chief of Staff reads calendar, tasks, operating signals
2. Domain agents report key changes
3. Chief of Staff produces morning brief
4. Evening review updates signals and next actions

Weekly loop:
1. Finance review
2. Family/home review
3. Professional/KRTA/project review
4. Infra/security review
5. Chief of Staff rebalances priorities

Execution principle:
- plan
- execute
- review
- verify
- compound
- repeat
