# Startup Playbooks

## Morning startup
Run order:
1. Chief of Staff Agent
2. Finance Agent
3. Family Operations Agent
4. KRTA Strategy Agent
5. Professional Strategy Agent

Purpose:
- gather overnight signals
- find hard deadlines and blockers
- compress into top 3 priorities
- identify one thing to defer or drop

Expected outputs:
- morning brief
- urgent signal list
- top 3 actions
- one deferral recommendation

## Evening startup
Run order:
1. Chief of Staff Agent
2. Finance Agent
3. Family Operations Agent
4. Device / Security / Infra Agent

Purpose:
- review completion
- update signals
- catch tomorrow risks
- reset next actions

Expected outputs:
- evening review
- tomorrow risk list
- carry-over actions

## Weekly strategy startup
Run order:
1. Chief of Staff Agent
2. Finance Agent
3. Professional Strategy Agent
4. KRTA Strategy Agent
5. Venture Studio Agent
6. Device / Security / Infra Agent
7. Family Operations Agent
8. Home & Purchasing Agent

Purpose:
- rebalance portfolio
- review finances and major commitments
- align family/home/professional systems
- promote or demote ideas

Expected outputs:
- weekly operating review
- risk register updates
- project priority changes
- new compound candidates
